### [Kakoune]

[Kakoune]: https://kakoune.org

#### Install

Add [`dracula.kak`] to your [`colors`] directory.

[`colors`]: https://github.com/mawww/kakoune#color-schemes
[`dracula.kak`]: https://github.com/dracula/kakoune/blob/master/colors/dracula.kak

#### Activating theme

``` kak
colorscheme dracula
```
